# DocTT - Frontend

[![Build Status](https://api.travis-ci.org/doctt/doctt-frontend.svg?branch=master)](https://travis-ci.org/doctt/doctt-frontend)

The frontend for the [DocTT](https://github.com/doctt) project.

## Requirements
- NPM (6.8.0+)
- Webpack CLI

## Run
- `npm install`
- `npm run build:dev`
- Visit http://127.0.0.1:8080/
